## Projeto em destaque

### Sistema de Gestão Operacional Logística

Plataforma web desenvolvida como projeto pessoal, inspirada em necessidades reais de operações logísticas e centros de distribuição.

**Principais recursos:**

- Dashboard operacional com indicadores e acompanhamento de resultados;
- Controle das operações de armazém e expedição;
- Acompanhamento PA–Cross com comparação entre planejado e realizado;
- Controle de pallets e processos de DRP;
- Cadastro, edição, consulta e exclusão de registros;
- Autenticação e controle de usuários;
- Integração com banco de dados PostgreSQL;
- Importação, tratamento e exportação de dados do Excel;
- APIs REST para integração entre módulos;
- Calculadora de recebimento para automatização dos custos de descarga.

**Tecnologias:** Python, Flask, JavaScript, HTML5, CSS3, PostgreSQL, Excel e APIs REST.

**Status:** projeto em evolução e preparação para publicação.
